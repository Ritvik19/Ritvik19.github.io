This Library helps in Solving Sudoku Puzzles using Artificial Neural Networks

Installation 

pip install ai-sudoku-solver

Usage

Instantiate a SudokuSolver object

                                      from ai_sudoku_solver import SudokuSolver
                                      solver = SudokuSolver("Ritvik19/sudoku-net-v1")

Call the model on your puzzles

                        puzzle = np.array([[
                            [0, 0, 4, 3, 0, 0, 2, 0, 9],
                            [0, 0, 5, 0, 0, 9, 0, 0, 1],
                            [0, 7, 0, 0, 6, 0, 0, 4, 3],
                            [0, 0, 6, 0, 0, 2, 0, 8, 7],
                            [1, 9, 0, 0, 0, 7, 4, 0, 0],
                            [0, 5, 0, 0, 8, 3, 0, 0, 0],
                            [6, 0, 0, 0, 0, 0, 1, 0, 5],
                            [0, 0, 3, 5, 0, 8, 6, 9, 0],
                            [0, 4, 2, 9, 1, 0, 3, 0, 0]
                        ]])
                        solution = solver(puzzle)
